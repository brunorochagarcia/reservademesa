# **App Name**: Seatly

## Core Features:

- Seat Layout Display: Visually represent the 21 seats in a 3x6, 3x6, 3x3 layout.
- Seat Selection: Allow users to select available seats.
- Reservation Confirmation: Provide a confirmation message after successful reservation.
- Seat Availability: Dynamically update seat availability status.

## Style Guidelines:

- Primary color: Dark indigo (#3F51B5), conveying professionalism and order.
- Background color: Light grey (#ECEFF1), providing a clean, unobtrusive backdrop.
- Accent color: Teal (#009688), used to highlight selectable and selected seats.
- Body and headline font: 'Inter', sans-serif font offering readability and a clean interface.
- Visually represent the 21 seats divided in 3 rows according to your specification.
- Use clear and simple icons to indicate seat availability and selected seats.
- Provide subtle animations for seat selection and reservation confirmation.
